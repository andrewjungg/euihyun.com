Arduino Black Jack

Used the Arduino and LED Matrix to create a simulation of Black Jack. Coded in C++. Link to the source code is here!

Macros were used to display the suit and number of the cards on the LED matrix. Interrupt routines were also utilized to allow the use of buttons.

The first button in the video was 'hit' which told the program (or the dealer) to deal out another card.

The second button was 'stay' which then gave you the sum of your cards. If you hit 21, the display would read 'BLACKJACK' on the matrix.

Samsung Galaxy S4:
UI Redesign

Keeping the idea of a universal design, I used specific human factors methods and design principles to analyze the UI of the Samsung Galaxy S4's accessibility mode. Following the completion of the first phase I created a redesign to better meet user requirements.

A feature that was added includes a pulldown menu button allowing you to turn on accessibility mode without travelling through the settings, giving it a simple mapping for users who utilize this mode.
